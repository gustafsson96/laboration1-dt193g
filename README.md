# Laboration 1 - Fullstacksutveckling med ramverk
<br>
Detta repository innehåller min kod för laboration 1 i kursen "Fullstacksutveckling med ramverk" hos Mittuniversitetet. Syftet med uppgiften är att bekanta sig med två CSS-ramverk för att sedan kunna välja ett att fördjupa sina kunskaper inom längre fram i kursen. Webbplatsen ska även publiceras med en plattform som inte är FTP-baserad. 
<br>

## Ramverk 
* index.html - Bootstrap (CDN)
* bootstrap.html - Bootstrap (CDN)
* tailwind.html - Tailwind CSS (CLI)
* deployment.html - Tailwind CSS (CLI)
